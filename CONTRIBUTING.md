# Contributing

## Read-only portfolio

This repository is a **personal portfolio / proof-of-concept**.  
It is maintained as **read-only** for visitors and is not open to direct contributions, pull requests, or issues aimed at changing the codebase.

### You may

- **Clone** the repository and run it locally
- **Use** the structure and patterns as inspiration for your own projects
- **Fork** for your own experiments (no expectation of upstream merges)

### We do not accept

- Pull requests
- Issue reports for feature requests or bugs (this is a static POC)
- Proposals to add or change functionality
